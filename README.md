# DownAnyVid

Professional Video & Audio Downloader built with React and Node.js.

## Supports:
- YouTube
- TikTok (No Watermark)
- Instagram
- Facebook
- Twitter/X
- 1000+ more sites (via yt-dlp)

## Setup:
### Backend
1. cd backend
2. npm install
3. npm start

### Frontend
1. cd frontend
2. npm install
3. npm run dev
